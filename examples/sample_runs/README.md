# Sample Runs

This directory contains example output from running the quant platform.

## Running a backtest

1. Build the project (see top-level README)
2. Run `QuantPlatform.exe` from the build directory
3. Click "Load Data" to load the sample OHLCV data
4. Click "Run Backtest" to execute the SMA Crossover strategy
5. Results will display in the output panel

## Console mode (Linux)

```bash
cd build
./QuantPlatform
```

The console mode automatically loads data, runs the backtest, and prints results.
